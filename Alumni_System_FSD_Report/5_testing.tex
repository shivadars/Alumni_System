\chapter{TESTING}

Software testing is an essential phase in the software development lifecycle, ensuring that the final product meets the defined requirements and is free of defects. In the Alumni Management System, a combination of manual and automated testing strategies was employed.

\section{Testing Principles}
Testing aims to identify bugs, verify the logic, and validate user experience. The primary objectives are:
\begin{enumerate}
    \item Uncover defects and logic errors in the code.
    \item Ensure that the system handles edge cases gracefully without crashing.
    \item Verify that unauthorized users cannot access protected routes.
    \item Validate the performance and responsiveness of the user interface.
\end{enumerate}
