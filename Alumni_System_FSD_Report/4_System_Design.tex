\section{Project Architecture}
The Alumni Management System follows the Model-View-Controller (MVC) architectural pattern:
\begin{itemize}
    \item \textbf{Model:} Eloquent classes define the core entities (User, Profile, Post, Comment, Message) and abstract database interactions.
    \item \textbf{View:} Blade templates combined with Tailwind CSS render the dynamic HTML interfaces presented to the users.
    \item \textbf{Controller:} Handles incoming HTTP requests, processes business logic, and returns the appropriate View or JSON response.
\end{itemize}
Laravel’s routing engine directs incoming traffic to specific controllers, while middleware ensures secure access control (e.g., verifying user roles or profile completion status before allowing access).

\section{Database Design}
Data storage is engineered relationally via PostgreSQL. The database schema consists of several core tables.

\subsection{Users Table}
The \texttt{users} table stores the central authentication details and role pointers for the system.

\begin{table}[H]
    \centering
    \small
    \begin{tabular}{|l|l|p{8cm}|}
        \hline
        \textbf{Field Name} & \textbf{Data Type} & \textbf{Description} \\
        \hline
        id & BigInt & Primary Key, Auto-increment \\
        \hline
        username & String & Unique username for login and display \\
        \hline
        email & String & Unique email address \\
        \hline
        password & String & Bcrypt Hashed password \\
        \hline
        role & String & User role (admin, alumni, student, department) \\
        \hline
        created\_at & Timestamp & Record creation date \\
        \hline
        updated\_at & Timestamp & Record last modification date \\
        \hline
    \end{tabular}
    \vspace{0.2cm}
    \caption{Users Table Schema}
    \label{tab:users_schema}
\end{table}

\begin{table}[H]

        user\_id & BigInt & Foreign Key referencing Users table \\

        \hline
        current\_job & String & Current employment title \\
        \hline
        profile\_picture & String & Path to the uploaded profile avatar \\
        \hline
    \end{tabular}
    \vspace{0.2cm}
    \caption{Profiles Table Schema}
    \label{tab:profiles_schema}
\end{table}
